# GOMS Discord Bot

광주소프트웨어마이스터고(GSM) 외출제 관리 서비스 **GOMS**와 연동되는 Discord 봇입니다.

외출제 관리 인원 모집, 참여자 수집, GOMS 서버 권한 동기화, 석식 안내 등을 자동화하여 학생회의 외출제 운영을 지원합니다.

## 주요 기능

### 외출제 관리 인원 모집

* 매주 월요일, 수요일 오전 9시 30분 자동 모집
* 모집 메시지 자동 전송
* 모집 메시지에 🙋 반응 자동 추가
* `@everyone` 멘션 지원

### 참여자 수집

* 매주 월요일, 수요일 오후 6시 20분 자동 수집
* 모집 메시지의 🙋 반응자 조회

### GOMS 서버 연동

* GOMS 내부 API 호출
* Discord 계정과 GOMS 계정 매핑
* 외출제 관리 인원 권한 자동 부여
* 기존 임시 권한 자동 초기화

### 동기화 결과 안내

* 동기화 결과 Discord 채널 전송

### 석식 안내

* 모집 참여자 자동 멘션
* 석식 식사 후 외출제 관리 위치 이동 안내

## 명령어

### 일반 명령어

| 명령어    | 설명               |
| ------ | ---------------- |
| `!명령어` | 사용 가능한 명령어 목록 표시 |

### 관리자 명령어

| 명령어        | 설명                        |
| ---------- | ------------------------- |
| `!외출제`     | 외출제 자동 발송 상태 확인           |
| `!외출제 on`  | 외출제 자동 발송 활성화             |
| `!외출제 off` | 외출제 자동 발송 비활성화            |
| `!외출제 모집`  | 모집 메시지 수동 전송 및 외출제 상태 활성화 |

## 동작 흐름

### 자동 모집

```text
09:30
↓
모집 메시지 전송
↓
🙋 반응 수집
```

### 참여자 수집 및 동기화

```text
18:20
↓
참여자 수집
↓
Discord User ID 추출
↓
GOMS 내부 API 호출
↓
권한 동기화
↓
결과 안내
↓
석식 안내
```

## 기술 스택

* Node.js
* Discord.js v14
* Axios
* Node Cron
* Dotenv
